### 📱 Aplicaciones Móviles (Mobile)

#### 📱 PrestaApp V2
> **Gestión integral de préstamos, finanzas personales y presupuestos.**

Una aplicación móvil moderna y segura, enfocada en la privacidad de los datos mediante encriptación local y herramientas financieras avanzadas.
- 🔒 **Seguridad de Grado Militar:** Base de datos encriptada con AES-256 (SQLCipher) y PIN de acceso.
- 📊 **Reportes y Finanzas:** Generación automática de reportes financieros en PDF y gestión de presupuestos mensuales dinámicos.
- 🛠️ **Stack Técnico:** ![Flutter](https://img.shields.io/badge/Flutter-02569B?style=flat&logo=flutter&logoColor=white) ![Riverpod](https://img.shields.io/badge/Riverpod-000000?style=flat&logo=dart&logoColor=white) ![SQLite](https://img.shields.io/badge/SQLite-003B57?style=flat&logo=sqlite&logoColor=white)

#### 🛡️ NullAds LTS (AdBlocker VPN)
> **Bloqueador de anuncios a nivel de sistema (DNS Sinkhole).**

Sistema híbrido que intercepta peticiones a rastreadores de publicidad en todo el dispositivo Android de forma global y altamente eficiente.
- 🧠 **Motor Heurístico Auto-Aprendizaje:** Algoritmo en Kotlin capaz de detectar y bloquear dominios sospechosos que no están en la base de datos (Zero-day ads).
- 🎨 **Laboratorio UI & Split Tunneling:** Temas dinámicos personalizables y capacidad de excluir aplicaciones bancarias del túnel VPN.
- 🛠️ **Stack Técnico:** ![Flutter](https://img.shields.io/badge/Flutter-02569B?style=flat&logo=flutter&logoColor=white) ![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat&logo=kotlin&logoColor=white) ![Android](https://img.shields.io/badge/Android_VpnService-3DDC84?style=flat&logo=android&logoColor=white)

#### 🎵 Quintet (TKOH Player)
> **Reproductor de música inteligente, inmersivo y dinámico.**

Una experiencia auditiva moderna construida con Flutter que reacciona visualmente a la música que estás escuchando en tiempo real.
- 🎨 **Temas Dinámicos Visuales:** La interfaz adapta sus colores automáticamente extrayendo la paleta dominante de la portada del álbum en reproducción.
- 🚀 **Rendimiento Fluido:** Optimizado con Riverpod para la gestión del estado y peticiones nativas rápidas a los archivos de audio local.
- 🛠️ **Stack Técnico:** ![Flutter](https://img.shields.io/badge/Flutter-02569B?style=flat&logo=flutter&logoColor=white) ![Dart](https://img.shields.io/badge/Dart-0175C2?style=flat&logo=dart&logoColor=white)

### 💻 Web & Frontend

#### 📚 TKOH Bookstore (Frontend)
> **Panel de administración avanzado (SPA) para gestión de librerías.**

Aplicación web moderna construida con React (Vite) diseñada para consumir la API `tkoh-bookstore-api`. Incluye autenticación con JWT, control de roles (RBAC) y un sistema de chat/notificaciones en tiempo real vía WebSockets.
- 🔐 **Gestión de Sesiones y Roles:** Flujo de autenticación completo (login, registro, OTP) y protección de rutas dinámicas basadas en los roles de usuario.
- 📡 **Reactividad e Interfaz:** Consumo eficiente del backend con React Query y Zustand, todo presentado bajo una interfaz impecable con Tailwind CSS y Shadcn/UI.
- 🛠️ **Stack Técnico:** ![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react&logoColor=61DAFB) ![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=flat&logo=typescript&logoColor=white) ![Vite](https://img.shields.io/badge/Vite-B73BFE?style=flat&logo=vite&logoColor=FFD62E) ![TailwindCSS](https://img.shields.io/badge/Tailwind-38B2AC?style=flat&logo=tailwind-css&logoColor=white) ![React Query](https://img.shields.io/badge/React_Query-FF4154?style=flat&logo=reactquery&logoColor=white)
